============
Installation
============


Stable release
--------------

To install kolla_dockerhub_pusher, run this command in your terminal:

.. code-block:: console

    $ pip install kolla_dockerhub_pusher

This is the preferred method to install kolla_dockerhub_pusher, as it will always install the most recent stable release. 

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for kolla_dockerhub_pusher can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/inc0/kolla_dockerhub_pusher

Or download the `tarball`_:

.. code-block:: console

    $ curl  -OL https://github.com/inc0/kolla_dockerhub_pusher/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/inc0/kolla_dockerhub_pusher
.. _tarball: https://github.com/inc0/kolla_dockerhub_pusher/tarball/master
